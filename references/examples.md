## 반복 업무 카테고리

Phase 2에서 사용. 참석자가 해당되는 항목 고르도록 유도.

```
혹시 이런 것들 중에 해당되는 거 있으세요?

**문서/보고 관련**
- 매주 같은 양식으로 보고서 쓰기
- 회의록 정리해서 공유하기
- 데이터 취합해서 표 만들기

**데이터/분석 관련**
- 여러 소스에서 숫자 모아서 정리하기
- 지표 확인하고 리포트 만들기
- 스프레드시트 데이터 가공하기

**커뮤니케이션 관련**
- 비슷한 내용 여러 사람에게 전달하기
- 진행 상황 업데이트 공유하기
- 피드백 요청 메시지 보내기

**기획/관리 관련**
- 일정 조율하고 캘린더에 등록하기
- 이슈/태스크 정리하고 추적하기
- 체크리스트 만들고 확인하기

이 중에 "어, 나 이거 맨날 해!" 싶은 거 있으세요?
없으면 다른 거 말씀해주셔도 돼요!
```

## PM 예시

Phase 4에서 사용. PM이 공감할 자동화 사례.

```
비슷한 업무를 자동화한 사례 몇 개 보여드릴게요:

**회의록 자동 정리**
"회의록 정리해줘"라고 하면:
- 회의 참석자, 날짜 자동 입력
- 논의 내용 → 결정사항/액션아이템으로 구조화
- Notion/Slack 양식에 맞게 포맷팅

**주간 보고서 자동 생성**
"주간보고 작성해줘"라고 하면:
- 이번 주 완료 태스크 자동 취합
- 정해진 양식에 맞게 정리
- 다음 주 계획까지 초안 작성

**스프레드시트 데이터 요약**
"이번 달 지표 정리해줘"라고 하면:
- 여러 탭에서 핵심 수치 추출
- 전월 대비 변화 계산
- 보고용 요약 텍스트 생성

혹시 이 중에 "오 이거 나도 필요해!" 싶은 거 있어요?
```

## Growth/마케팅 예시

Phase 4에서 사용. Growth PM/마케터가 공감할 자동화 사례.

```
Growth PM이시면 이런 것들 어떠세요?

**캠페인 성과 리포트**
"어제 캠페인 성과 정리해줘"라고 하면:
- 채널별 지표 자동 취합
- 전일/전주 대비 비교
- 인사이트 코멘트 초안까지

**콘텐츠 캘린더 관리**
"이번 주 콘텐츠 현황 알려줘"라고 하면:
- 예정된 포스팅 목록
- 진행 상태 (작성중/검수중/발행완료)
- 빠진 것 알림

**경쟁사 모니터링**
"경쟁사 최근 소식 정리해줘"라고 하면:
- 설정한 경쟁사 SNS/뉴스 체크
- 주요 변화 요약
- 우리 대응 필요 여부 코멘트
```

## 디자이너 예시

Phase 4에서 사용. 디자이너가 공감할 자동화 사례.

```
디자이너시면 이런 건 어떠세요?

**디자인 핸드오프 문서**
"개발 전달용 정리해줘"라고 하면:
- 컴포넌트별 스펙 정리
- 인터랙션 설명
- 체크리스트 생성

**피드백 정리**
"디자인 리뷰 피드백 정리해줘"라고 하면:
- 흩어진 코멘트 한 곳에 모으기
- 수정 필요/검토 필요/완료 분류
- 우선순위 제안

**디자인 시스템 문서화**
"이 컴포넌트 문서 만들어줘"라고 하면:
- 사용 가이드라인
- Do/Don't 예시
- 변형(variants) 정리
```
